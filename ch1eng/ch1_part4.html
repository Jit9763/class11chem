Part 4: Uncertainty in Measurement (मापन में अनिश्चितता)

4.1 Scientific Notation (वैज्ञानिक संकेतन)

Expressing chemistry values in the form of N × 10ⁿ is called 'Scientific Notation'. (Where N = between 1.000... and 9.999... and n = exponent).

Example: Avogadro No.

6.022 × 10²³

Example: H-Atom Mass

1.66 × 10⁻²⁴ g

4.2 Significant Figures (सार्थक अंक)

"When the result of a measurement is obtained accurately, the number of meaningful digits in that measurement is called 'Significant Figures'."

📐 Rules for Finding Significant Figures:

Rule 1

All non-zero digits are significant.

(a) 2478 → 4
(b) 34125 → 5
Rule 2

All zeros appearing between two non-zero digits are significant.

(a) 2024 → 4
(b) 90205 → 5
Rule 3

Zeros appearing to the left of the first non-zero digit are NOT significant.

(a) 0.00486 → 3
(b) 0.01 → 1
Rule 4

In a number without a decimal, zeros to the right of the last non-zero digit are NOT significant.

(a) 4500 → 2
(b) 2050 → 3 (2, 0, 5)
Rule 5

In a number with a decimal, zeros to the right of the last non-zero digit ARE significant.

(a) 2.00 → 3
(b) 1230.0 → 5
Rule 6

The number of significant figures in Counting of objects is Infinite (∞).

(a) 2 balls → ∞
(b) 50 eggs → ∞

➡️ Rounding Off (सनिक्वटन करना)

1. If the digit to be removed is less than 5: No change is made to the preceding digit.
Ex: 12.862 → 12.86 | 4.324 → 4.32
2. If the digit to be removed is greater than 5: Add +1 to the preceding digit.
Ex: 12.868 → 12.87 | 4.329 → 4.33
3. If the digit to be removed is "5":
(a) If preceding digit is Odd: Add +1
(b) If preceding digit is Even: Keep it unchanged
Ex: 4.25 (Even) → 4.2 | 27.75 (Odd) → 27.8

4.4 Significant Figures in Calculations

➕ Addition & Subtraction:

The result should have as many digits after the decimal as the number with the least decimal places.

12.11 + 18.0 + 1.012 = 31.122
Final → 31.1 (w.r.t 18.0)
✖️ Multiplication & Division:

The result should have as many total significant figures as the number with the least significant figures.

Density = 4.237 / 2.51 = 1.6880...
Final → 1.69 (w.r.t 2.51: 3 SF)
